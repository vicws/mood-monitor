# mood-monitor
ptBR
Este é um diário online para monitoramento da saúde mental.
No momento, pode ser usado para registrar seu humor, e as entradas ficam salvas no localStorage.
Foi construído com ajuda de IA, utilizando JS, HTML e CSS. Toda lógica do produto foi criada por mim.
